Social-linker

Angularjs directive that binds scope-driven social buttons! Even includes link shrinker for twitter!

Download .zip Download .tar.gz View on GitHub

social-linker v0.0.2

How to use:/h3>
  1. Add jQuery
  2. Include the directive script.
  3. Include the module social-linker in your app module.
  4. Import and add a socialLinkerOpts object to your rootScope (had to use it to avoid prototypical inheritence issues)
  5. Add each of your social objects to the socialLinkerOpts object.

Like:

$scope.socialLinkerOpts = { OR $rootScope.socialLinkerOpts = { 
 watch : ["/posts/","/IdontWantIthere/"],
 locationChange : true,
 twitter : {
 title : $scope.title,
 description : $scope.subTitle,
 url : "http://www.awesomesite.com/post/"+$scope.postURL,
 },
 facebook : {
 title : $scope.title,
 description : $scope.subTitle,
 url : "http://www.awesomesite.com/post/"+$scope.postURL,
 image : "http://www.sweetcoolthing.com/image1.png"
 }
}

Properies

watch
ARRAY GLOBAL an array of routes you'd like social-linker to update on, leave this out to check ALWAYS 
locationChange
BOOL GLOBAL want this to update on route change? No? Make this false. 
title
STRING Title of the page to be shared 
description
STRING Description of page to be shared 
image
STRING an image of what will be shared 
url
STRING A link to the page that will be shared
